# Altrompy

**Altrompy** (a play on *Entropy*) is the customized, professional-grade boilerplate used as the foundation for all **Altron-generated** projects. It is built on the Laravel/Livewire Starter Kit and enhanced with Flux Pro components, providing a battle-tested starting point for modern SaaS applications.

## üõ† Usage

This repository is primarily used by the Altron orchestration engine to scaffold new projects. However, it can also be used manually:

1.  **Clone the repo**: `git clone https://github.com/jdecode/altrompy.git my-new-app`
2.  **Install dependencies**: `composer install` & `npm install`
3.  **Setup environment**: Copy `.env.example` to `.env` and configure your database.
4.  **Generate key**: `php artisan key:generate`
5.  **Run migrations**: `php artisan migrate`
6.  **Start development**: `npm run dev`
