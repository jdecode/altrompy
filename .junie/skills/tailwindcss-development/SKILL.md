# Tailwind CSS v4
CSS-first configuration.

## Documentation
Use `search-docs`.

## Usage
- **Config**: `@theme` in CSS, no `tailwind.config.js`.
- **Import**: `@import "tailwindcss"`.
- **Values**: Use numeric opacity (e.g. `bg-black/50`).
- **Core**: No `corePlugins`.
